Tema 1: Etiquetas Basicas de Html

Html esta compuesto por Etiquetas que marcan el inicio y el final de cada elemento del documento. Html te permite crear contenidos Multimedia y su visualizacion. Deberias utilizar Html para crear tus paginas, sin importar el tipo de contenido que quieres tener en ella. Tambien es importante señalar que las paginas Html son simples archivos de textos. El texto es la manera mas corriente de introducir datos en un ordenador o un navegador

Tema 2: Atributos y Valores

Los elementos en HTML tienen atributos; estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios. En HTML, la mayoria de los atributos tiene dos caras: el atributo de contenido y el atributo IDL El atributo de contenido es aquel que se define desde el contenido (Codigo HTML) y puedes definirlo u obtenerlo con element.setAttribute()o element.getAttribute() El atributo de contenido es siempre un string incluso cuando el valor esperado en un integer. El atributo IDL, también conocido como propiedad JavaScript. Estos son atributos que se pueden leer o definir usando JavaScript como (element.foo). El atributo IDL siempre va a usar (pero puede transformarse) en el atributo de contenido subyacente.
